# 🌟 Personal Portfolio Website

A modern, responsive, and professional **Developer Portfolio Website** built using HTML, Sass, Bootstrap, Tailwind CSS, JavaScript, and EmailJS.  
Packed with smooth animations, a chatbot, and deployed using GitHub Pages via Vite bundler for fast performance.

---

## 🚀 Features

- ✨ Clean and modern UI design  
- 📱 Fully responsive on all devices  
- 💬 Built-in chatbot using pure JavaScript  
- 📧 Contact form integrated with **EmailJS**  
- 🎨 Styling with **Sass**, **Tailwind CSS**, and **Bootstrap**  
- ⚡ Super-fast builds with **Vite**  
- 🌐 Live hosted on GitHub Pages

---

## 🛠️ Technologies Used

- `HTML5` – Semantic structure  
- `Sass (SCSS)` – Custom reusable styles  
- `Bootstrap` – Responsive grid and UI utilities  
- `Tailwind CSS` – Utility-first CSS framework  
- `JavaScript (Vanilla)` – DOM manipulation and interactivity  
- `EmailJS` – For sending form data via email  
- `Vite` – Lightning-fast dev server and build tool  
- `AOS` – Animate on Scroll library for smooth animations  
- `GitHub Pages` – Deployment
